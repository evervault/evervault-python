# Contributing

Bug reports and pull requests are welcome on GitHub at https://github.com/evervault/evervault-python.

## Getting Started

- `poetry install`
- `poetry run pytest`
- `poetry run black .`
- `poetry run flake8 --ignore=E501,W503`
